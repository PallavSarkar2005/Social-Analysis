# Social Analysis

AI-powered social media analytics platform that helps creators, marketers, and businesses analyze YouTube channels and X (Twitter) accounts, track engagement metrics, compare performance, and generate AI-driven insights.

---

## Features

### YouTube Analytics

* Channel performance tracking
* Subscriber and view analytics
* Video performance insights
* Top-performing videos analysis
* Growth trend visualization

### X (Twitter) Analytics

* Profile analysis
* Engagement metrics
* Follower growth tracking
* Account comparison
* Content performance insights

### AI-Powered Insights

* Automated account analysis
* Content strategy recommendations
* Channel health scoring
* Performance summaries
* Growth opportunity detection

### Dashboard & Visualization

* Interactive charts
* KPI monitoring
* Engagement reports
* Historical tracking
* Multi-account comparison

## Installation

### Clone Repository

```bash
git clone https://github.com/PallavSarkar2005/Social-Analysis.git
cd Social-Analysis
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGODB_URI=your_mongodb_uri
OPENAI_API_KEY=your_api_key
```

Start Backend:

```bash
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
